# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.3.3] - 2026-05-21

### Added
- Comprehensive README with full documentation
- Development guide for contributors
- Security policy and best practices
- Code of Conduct for community
- Issue and PR templates for GitHub

### Improved
- Enhanced documentation with more examples
- Added security guidelines
- Better project structure documentation
- Clearer installation instructions

## [1.3.3] - 2011-04-19

### Added
- UTF-8 encoding support
- Multi-fragment processing via Albino::Multi
- Custom encoding configuration
- Timeout threshold configuration

### Fixed
- Markdown output formatting
- Shell argument validation for security

## [1.3.0]

### Added
- POSIX::Spawn integration for process management
- Improved error handling
- Encoding support for various formats

## [1.0.0] - Original Release

### Added
- Basic syntax highlighting wrapper
- Support for multiple output formats
- File and string input handling
- Configurable lexers and formatters
